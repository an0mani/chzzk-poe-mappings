PoE Viewer for Chzzk - 개인정보처리방침

시행일: 2026년 3월 2일

1. 수집하는 개인정보

본 확장프로그램은 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

2. 데이터 처리

본 확장프로그램은 다음 데이터를 처리하지만, 모두 사용자의 브라우저 내에서만 처리되며 외부 서버로 전송되지 않습니다:

  • 캐시 데이터: Path of Exile API 응답을 브라우저 로컬 스토리지에 일시 캐시합니다 (최대 5분).
  • 사용자 설정: 새로고침 간격 등 설정값을 브라우저 로컬 스토리지에 저장합니다.

3. 외부 서비스 통신

본 확장프로그램은 다음 외부 서비스와 읽기 전용으로 통신합니다:

  • pathofexile.com: 공개 프로필의 캐릭터 및 장비 데이터 조회
  • api.chzzk.naver.com: 방송 카테고리 (게임 종류) 확인
  • raw.githubusercontent.com: 스트리머-캐릭터 매핑 데이터 조회

모든 통신은 HTTPS를 통해 이루어지며, 사용자 식별 정보는 전송되지 않습니다.

4. 쿠키 및 추적

본 확장프로그램은 쿠키를 사용하지 않으며, 사용자 행동을 추적하지 않습니다. 분석 도구나 광고 SDK를 포함하지 않습니다.

5. 제3자 제공

본 확장프로그램은 어떠한 데이터도 제3자에게 판매, 제공, 전송하지 않습니다.

6. 권한 사용 목적

  • storage: API 응답 캐시 및 사용자 설정 저장
  • alarms: 주기적 캐시 정리
  • host_permissions: Path of Exile API, Chzzk API, GitHub에서 공개 데이터 읽기

7. 변경 사항

본 방침이 변경될 경우, 이 페이지를 통해 공지합니다.
